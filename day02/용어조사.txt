< 구글링 조사 >

- 서버: 클라이언트에 데이터 응답
- 클라이언트: 서버로부터 데이터 요청
- 요청: 클 -> 서
- 응답: 서 -> 클
- 네트워크: 기기 간의 통신망
- 아이피: Internet Protocol cf) IP address
- 포트: 프로그램 간의 가상 데이터 연결망
- 방화벽: 네트워크 선택적 차단벽
- URI: Uniform Resource Identifier
- URL: URI 하위 개념, Uniform Resource Locator
- URN: URI 하위 개념, Uniform Resource Name
- 게이트웨이: 네트워크 통신 경유지
- 라우터 / 라우터스위치: 게이트웨이 역할을 하는 기기
- 프록시: 네트워크 요청 중개자
- LAN / WAN: Local Area Network, Wide Area Network
- www: World Wide Web
- 인터넷: 세계적으로 연결된 TCP/IP 네트워크망
- 패킷: 네트워크 상에 전송되는 데이터 조각
- 소켓: 네트워크 통신 끝자락
- 프로토콜: 네트워크 통신 규약
- TCP: Transmission Control Protocol
- UDP: User Datagram Protocol
- VPN: Virtual Private Network
- MAC Address: Media Access Control address
- DNS: Domain Name System
